# AWS Monitoring and Auditing Lab

## Overview
This project demonstrates monitoring, observability, and auditing techniques in AWS using CloudWatch, CloudTrail, SNS, and Amazon S3.

## Objectives
- Launch an EC2 instance
- Configure CloudWatch alarms
- Create SNS notifications
- Monitor CPU utilization
- Store CloudTrail logs in S3
- Explore auditing and observability concepts

## Technologies
- Amazon EC2
- Amazon CloudWatch
- Amazon SNS
- AWS CloudTrail
- Amazon S3

## Skills Demonstrated
- Cloud monitoring
- Operational observability
- Event auditing
- Alert automation
- Log management
- Troubleshooting in restricted environments
